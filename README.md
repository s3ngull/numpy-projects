# NumPy Projects

Welcome to **NumPy Projects**, a collection of small projects demonstrating **NumPy usage in biomedical simulations and data analysis**.

## About This Repository

This repository contains beginner-friendly projects created using **Python and NumPy**. Each project showcases different applications of NumPy, including:

- Simulating blood test data and analyzing hemoglobin levels
- ECG signal simulation and analysis
- Basic medical image thresholding
- Linear regression using NumPy
- More projects will be added over time

The projects are designed to:

- Strengthen understanding of **NumPy arrays, statistics, and logical operations**
- Provide **hands-on biomedical examples**
- Be easily understandable and modifiable for beginners
